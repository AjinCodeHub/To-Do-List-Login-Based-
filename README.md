Login-Based To-Do List App (Python)

A simple **Command Line To-Do List Application** built using **Python**.
This application allows users to create an account, log in, and manage their personal tasks.

Each user has their **own separate task list**, making it a basic **multi-user task management system**.

---

## 🚀 Features

* User Account Creation
* User Login Authentication
* Add Tasks
* View Tasks
* Remove Tasks
* Multi-User Support
* Password Length Validation
* Simple CLI (Command Line Interface)

---

## ⚙️ How It Works

### 1️⃣ Create Account

The user creates an account with:

* Username
* Email
* Password (minimum 8 characters)

Example:

```
1. Create Account
Enter username: username
Enter email: username@email.com
Enter password: ********
```

---

### 2️⃣ Login

Users log in using their **username and password**.

```
2. Login Account
Enter username: useranme
Enter password: ********
```

If credentials are correct, the **To-Do menu** will open.

---

### 3️⃣ To-Do Menu

```
1 - Add Task
2 - Display Tasks
3 - Remove Task
4 - Logout
```

Users can manage their personal tasks from this menu.
